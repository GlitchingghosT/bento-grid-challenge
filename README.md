# Bento Grid Challenge

A responsive Frontend Mentor bento-grid implementation for presenting social-media product metrics in an asymmetric card layout.

- **Live demo:** https://bento-olive.vercel.app
- **Source:** https://github.com/GlitchingghosT/bento-grid-challenge

## Features

- Responsive bento-style CSS grid
- Reordered card composition across desktop and mobile
- Local DM Sans font assets
- Accessible semantic content structure

## Built with

HTML and Tailwind CSS.

## Run locally

No build step is required. Clone the repository and open `index.html` in a browser. Tailwind is loaded through the CDN.
